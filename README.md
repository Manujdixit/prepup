# Tayyari Plus

A modern learning and assessment system.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/tayyari_plus.git
   cd tayyari_plus
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
